ssh登陆远程主机

###方法一：
将.ssh/id_rsa.pub内容加入到远程主机.ssh/authorized_keys下

注意文件是authorized_keys 不是authorize_keys

###方法二:

将id_rsa.pub自动添加到authorized_keys尾部.

文件名必须是id_rsa.pub

###attention
.ssh文件夹权限必须是700

.ssh/authorized_keys必须是600

无效时清理下主机.ssh/known_hosts
